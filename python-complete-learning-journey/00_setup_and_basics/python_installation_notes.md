# Python Installation Notes

- Recommended: install latest stable Python 3.10+ or 3.11
- Use virtual environments (venv / pipenv / poetry)

Setup check:
- python --version
- pip --version

Quick tip: Use `python -m venv .venv` and `.\.venv\Scripts\activate` on Windows.
