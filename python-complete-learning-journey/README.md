# Python Complete Learning Journey

This repository is structured as a progressive learning path for Python beginners. Each folder contains small example scripts and practice problems to help you learn and demonstrate knowledge.

Structure overview:
- 00_setup_and_basics
- 01_control_flow
- 02_strings
- 03_lists_tuples_sets
- 04_dictionaries
- 05_functions
- 06_oops
- 07_exception_handling
- 08_file_handling
- 09_modules_and_packages
- 10_numpy
- 11_pandas
- 12_problem_solving

How to use:
- Run small scripts (e.g., `python 00_setup_and_basics/variables_and_datatypes.py`) for examples.
- Each script includes practice problems in comments.

Requirements:
- Common packages: numpy, pandas

Contributions:
- Welcome! Create PRs for new exercises or solutions.

Practice index: Each module contains at least 2 practice tasks (see top of each file).
